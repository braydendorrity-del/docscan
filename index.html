⚙️ One-time Drive setup

To upload to your Google Drive, you need a free API key. Here's how:

  1. Go to console.cloud.google.com
  2. Create a project → Enable Google Drive API
  3. Go to APIs & Services → Credentials
  4. Create an OAuth 2.0 Client ID → Web application
  5. Add this page's URL to Authorized JavaScript origins
  6. Paste your Client ID below and tap Save
